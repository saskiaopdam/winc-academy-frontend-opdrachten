Opleiding

Front-end development

HTML

De inhoud van een website wordt beschreven in HTML. Je bepaalt hiermee welke pagina's er zijn en wat waar staat. Je leert HTML lezen en schrijven, en kunt hiermee zelf eenvoudige websites maken.

CSS

Met CSS geven we onze website een likje verf. CSS gaat over het design en alles omtrent het geven van een aantrekkelijk uiterlijk aan je website. Denk aan kleuren, lettertypes, animaties, etc.

JavaScript

In dezelfde vergelijking is JavaScript de elektriciteit die alles doet werken. Met JavaScript voegen we functionaliteit toe aan onze website. Interactie met gebruikers, databases, animaties, etc.

React

React is een 'uitbreiding' op JavaScript, waarmee we nog makkelijker snelle, soepel werkende websites leren schrijven. Websites als Facebook, Netflix, en Reddit zijn allemaal in React ontwikkeld!

Testing & workflow

Ten slotte leer je tests schrijven, die je helpen bij het maken van een goed werkende, stabiele website die nooit kuren heeft! Ook leer je developer workflows m.b.v. Git en andere nuttige tools.

Back-end development

Python

Python is een van de meest populaire programmeertalen ter wereld. Je leert in deze taal lezen en schrijven tot het punt dat je er zelfstandig een uitgebreide applicatie mee kan bouwen.

Flask

Flask is een verlengstuk voor Python, waarmee je zelf applicaties in de cloud kunt bouwen. Door Flask te leren gebruiken begrijp je ook de fundamentele werking van het internet.

Databases

Een van de belangrijkste aspecten van de cloud is de opslag en het gebruik van data. Je leert werken met de database-taal SQL en je leert databases te gebruiken vanuit Python.

Deployment

Deployment gaat over het online zetten van webapplicaties. Je leert wat de cloud inhoudt, hoe je daar zelfgeschreven software op installeert en hoe je die daarna onderhoudt.
